Getting Started
-------------

If you're new to The Aichemist Codex, we recommend starting with the :doc:`quick_start` tutorial, which will guide you through the basic setup and usage of the system.

Once you're familiar with the basics, you can explore more advanced features:

- :doc:`tagging_workflow`: Learn how to use the powerful tagging system for organizing and classifying files
- :doc:`semantic_search`: Master advanced semantic search techniques for finding conceptually related content
- :doc:`file_organization`: Discover automated file organization and structuring capabilities
- :doc:`advanced_api_usage`: Integrate The Aichemist Codex into your custom applications
- :doc:`custom_plugins`: Extend functionality with custom plugins and integrations
- :doc:`file_relationships`: Understand how to map and visualize relationships between files
- :doc:`search_techniques`: Learn advanced search techniques beyond semantic search

Contributing New Tutorials
------------------------

If you'd like to contribute a new tutorial, please follow these steps:

1. Create a new `.rst` file in the `docs/tutorials/` directory
2. Follow the existing tutorial format and style
3. Include practical examples and code snippets
4. Add your tutorial to the toctree in this index file
5. Submit a pull request with your changes

We welcome contributions that help other users discover and master The Aichemist Codex!
